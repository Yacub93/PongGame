# Pong Game App

Pong game developed using JavaScript with the Phaser HTML5 Game Framework.


Link to Framework site: https://phaser.io

## Features
 * Game Scores recorded
 * Paddle 2 controlled simple AI 
 * Includes paddle sound effects
